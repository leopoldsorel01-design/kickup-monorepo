# KickUp! Monorepo

This is the monorepo for the **KickUp!** platform.

## Structure

*   **apps/mobile**: React Native Application.
*   **apps/api**: GraphQL API Gateway.
*   **packages/drill-engine**: XState FSM Logic.
*   **packages/ui-kit**: Shared UI Components.

## Getting Started

1.  Install dependencies: `yarn install`
2.  Run dev: `yarn dev`
